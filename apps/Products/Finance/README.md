# Finance Module

This is the Finance business module for the Saudi Store platform. This module provides comprehensive financial management capabilities including:

- Accounts receivable (A/R)
- Accounts payable (A/P)
- General ledger
- Chart of accounts
- Financial periods management
- Multi-currency support
- Billing & Invoicing
- Budget Management
- Financial Reporting
- Expense Management

## Structure

- `apps/web`: Frontend components and pages
- `apps/services`: Backend microservices
  - `api`: REST API endpoints
  - `schema`: Database schemas and migrations
- `docs`: Module documentation
- `contracts`: API contracts and specifications

## Integration

This module can be used independently or integrated with other DoganHubStore modules.
